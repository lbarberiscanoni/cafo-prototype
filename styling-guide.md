**Rules:**
- Prefer brand colors without editing
- When tints are necessary, use 20% increment steps
- Available tints: 100%, 80%, 60%, 40%, 20%
- Any tint below 60% used as background requires dark text for legibility
- Apply tinting system to all brand colors (primary and secondary)

**Example Tints for MTE Blue:**
- 100%: `#00ADEE` (full opacity)
- 80%: `rgba(0, 173, 238, 0.8)`
- 60%: `rgba(0, 173, 238, 0.6)`

## 2. TYPOGRAPHY

### Font Families

**Primary Typeface: Lato**
- Usage: Every brand execution, all body text, most headings
- Weights to use: Light (300), Regular (400), Medium (500), Heavy (900)
- Fallback: Helvetica, Arial, sans-serif (system defaults)
- Notes: Variable-weight typeface; use contrast between heavy and lighter weights to create hierarchy

**Secondary Typeface: Source Serif Pro**
- Usage: Subheadings, emphasis, stylistic variance
- Weights to use: Regular (400), SemiBold (600), Italic variations
- Fallback: Georgia, Times, serif (system defaults)
- Notes: Use for subheadings and to add variety; can use italics for emphasis

**Specialty Typeface: Nexa Rust Script S 00**
- Usage: Sparingly - "cherry on the cake" for special applications
- Specific uses: State seals, event signage, personal feel applications
- Weight: Regular (400) only
- Notes: NOT a primary or secondary typeface; use only for special emphasis

### Typography Hierarchy - Option 1 (Formal Structure)

**Heading One**
- Font: Lato Heavy (900)
- Size: 80px (4.44rem)
- Bottom Margin: 50px
- Usage: Page headings, major section headings, emphasized words

**Heading Two**
- Font: Lato SemiBold (600)
- Size: 52px (2.89rem)
- Bottom Margin: 30px
- Usage: Section headings, blog body headings, product headings

**Heading Three**
- Font: Source Serif Pro Italic
- Size: 34px (1.89rem)
- Bottom Margin: 30px
- Usage: Subheadings, call to action headings, blog subheadings

**Heading Four**
- Font: Lato Bold (700), UPPERCASE
- Size: 18px (1rem)
- Bottom Margin: 20px
- Usage: Minor headings, table labels, sub-sub headings

### Typography Hierarchy - Option 2 (Content Structure)

## 3. LAYOUT & COMPOSITION RULES

### The Six Rules of Type

**Rule 01: Stay Left-Aligned, Rag Right**
- Principle: Legibility and clarity are vital
- Implementation: Align text left since most people read left to right
- Text-align: left (default for body content)

**Rule 02: Skip Weights & Double Size**
- Principle: Contrast creates visual interest
- Implementation: 
  - When pairing font weights, skip a weight (e.g., Regular → Bold, not Regular → Medium)
  - When varying text sizes, double the size difference
  - Example: 18px body text → 36px heading (not 24px)

**Rule 03: Align X-Heights or Baselines**
- Principle: Visual alignment creates professional appearance
- Implementation: When placing text side by side, align either:
  - Baselines (bottom of lowercase 'x')
  - X-heights (top of lowercase 'x')

**Rule 04: Watch The Rag**
- Principle: Maintain clean right edge on left-aligned text
- Implementation:
  - Monitor the ragged right edge of paragraphs
  - Avoid unintentional recognizable shapes
  - Prevent orphans (single-word lines)
  - Adjust language or container width if needed

**Rule 05: Give Things Space, If Needed**
- Principle: Negative space is vital; grouping creates relationships
- Implementation:
  - Elements that belong together should be closer
  - Don't cram too many things in one space
  - Use white space deliberately

**Rule 06: Keep Line Length Reasonable**
- Principle: Readability depends on line length
- Implementation:
  - Keep lines between 45-70 characters long
  - Adjust based on font size
  - Too long: readers get lost
  - Too short: easily ignored

## 6. ACCESSIBILITY REQUIREMENTS

### Color Contrast
- Text below 60% tint requires dark text
- Body text should be Charcoal (#5c5d5f) or Black (#000000)
- Links must be MTE Blue (#00ADEE) with underline for accessibility

### Readability
- Line length: 45-70 characters
- Line height: Specified for each text style (see Typography section)
- Left-aligned text for readability

---

## 7. BRAND EXCEPTIONS & NOTES

### When Brand Fonts Are Not Available
- Lato fallback: Helvetica → Arial → sans-serif
- Source Serif Pro fallback: Georgia → Times → serif
- Never use Nexa Rust Script substitutes; simply omit if unavailable

### Color Usage Philosophy
- Prefer exact brand colors without modification
- Use tints only when necessary
- Secondary colors should be used sparingly
- MTE Blue is the hero - use prominently

### Breaking the Rules
- The six rules are proven and sound
- Breaking them occasionally is acceptable
- Consult design team when considering breaking guidelines
- Maintain brand integrity above all
